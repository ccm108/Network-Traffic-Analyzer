Network Traffic Analyzer

This project is a simple network traffic analyzer that was made to help monitor and inspect packets moving through a network. The program can capture traffic and display information like IP addresses, protocols, and packet activity. It helps give a better understanding of how network communication works and can also help identify suspicious traffic.

Features

- Captures network packets
- Shows source and destination IP addresses
- Displays protocol information
- Monitors traffic in real time
- Simple command-line program
- Can help spot unusual network activity

Requirements

- Python 3
- Libraries from requirements.txt
- Admin permissions may be needed to capture packets

Installation

Clone the repository

"git clone https://github.com/ccm108/Network-Traffic-Analyzer.git"

Open the project folder

"cd Network-Traffic-Analyzer"

Install the required libraries

"pip install -r requirements.txt"

How to Run

"python main.py"

After running the program it should start showing network traffic information in the terminal.

Example

"python main.py"

The program may show things like:

- IP addresses
- Packet types
- Protocols being used
- General network activity

Ethical Considerations

This project should only be used on networks you own or have permission to test on. Capturing traffic on random networks could violate privacy or security rules. This project was mainly made for educational purposes and learning more about networking and cybersecurity.
